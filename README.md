## Install

### Tree
- 파일명은 install.sh로 통일
- 경로는 기본적으로 'OS/버전/구성할 대상/install.sh' 형태
```
- ubuntu
    - jammy
        - apt
        - docker
        - go
    - noble
        - apt
        - docker
        - go
- etc
    - docker-compose
    - jenkins
```


### Commands
```bash
wget -O - <raw URL> | sh
# or
curl -fsSL <raw URL> | sh
```
